# Welcome to Portfolio 

A modern, minimal, and multi-page developer portfolio built using HTML, CSS, and Vanilla JavaScript.

Designed to showcase:
- projects
- technical skills
- open-source work
- communities
- personal identity

The website focuses on clean layouts, calm animations, structured hierarchy, and a premium dark-first experience without unnecessary visual noise.

---

## Features

- Multi-page architecture
- Premium dark/light theme system
- Responsive design
- Structured project showcase
- Community ecosystem page
- Animated but restrained UI
- SEO-ready structure
- Clean URL routing
- Contact form with backend protection
- Smooth page transitions
- Accessible and performance-focused design

---

## Design Philosophy

This project avoids:
- excessive animations
- visual clutter
- overengineered frameworks
- template-like UI

Instead, it prioritizes:
- readability
- hierarchy
- clarity
- subtle motion
- maintainability
- proof-of-work presentation
