File Organizer

An interactive command-line tool that cleans up directories and sorts files into sensible groups.

Overview

File Organizer takes a cluttered directory and turns cleanup into a guided, repeatable CLI workflow.

Problem Solved

Downloads and desktop folders become noisy quickly. This tool helps separate files by type and keeps the cleanup process understandable.

Screenshot

Architecture

  • Interactive CLI prompts for user control.
  • File type grouping and sorting rules.
  • Directory scanning before moving files.
  • Readable output for cleanup visibility.

Technical Decisions

The tool prioritizes transparency, so cleanup feels controlled instead of automatic in a risky way.

Challenges

File tools need careful defaults because moving user files should feel reversible, clear, and intentional.

Lessons Learned

Small desktop utilities benefit from the same design discipline as larger apps: clear states, cautious actions, and predictable structure.
